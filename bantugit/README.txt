Instructions to run BantuOS:
============================

1. First, you will nedd the g++ cross-compiler for running BantuOS. The exact instructions are given here: http://wiki.osdev.org/GCC_Cross-Compiler
2. Step 1 will generate a compiler by the name of i686-elf-g++. Make sure to add it in your search path.
3. Copy the source code in any empty folder.
4. Run `make` from that directory.
5. It will generate an iso file called bantu.iso. This is the image fil;e for our BantuOS. Run it in any virtual machine, say VirtualBox.
6. You are done!
